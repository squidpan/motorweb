# Job Posting User Stories

## Create Job Posting

As a recruiter,
I want to create a job posting,
So that applicants can see open positions.

## View Job Postings

As an applicant,
I want to view available jobs,
So that I can decide which jobs to apply for.

## Update Job Posting

As a recruiter,
I want to update a job posting,
So that job details stay accurate.

## Delete Job Posting

As a recruiter,
I want to delete a job posting,
So that closed or invalid jobs are no longer shown.
